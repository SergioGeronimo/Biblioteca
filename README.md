# Biblioteca
Proyecto escolar, ejemplo de una *bilbioteca básica* con SQL/XAMPP, HTML5 y PHP.

Usar este proyecto dentro de la carpeta (w10):
C:\xampp\htdocs\\**inserte proyecto**
